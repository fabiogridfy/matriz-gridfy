Escala de niveles

Cada skill se puntúa del 1 al 10. La guía de preguntas tiene anclas en los puntos 2, 5 y 8 — los demás puntos son interpolación libre del evaluador.

1–2
Muy bajo / Básico
Definición y reconocimiento. ¿Sabe qué es esto? ¿Lo ha visto?
3–4
Inicial
Nociones aplicadas. Puede seguir casos simples con apoyo.
5–6
Sólido
Aplicación y criterio. Resuelve casos reales con autonomía razonable.
7–8
Avanzado
Diseño y trade-offs. Gestiona complejidad y toma posición.
9–10
Excelente
Criterio experto. Visión propia, detecta lo que está roto.

Los botones marcados con borde superior (2 · 5 · 8) corresponden a las anclas con preguntas asociadas en la guía. Los demás puntos los interpola el evaluador.

🔗 Sincronización con Google Sheets INACTIVO

Permite que varios evaluadores compartan candidatos en tiempo real a través de Google Sheets.
Para activarlo: crea una Google Sheet + un Apps Script con el código de abajo, pega la URL del script aquí y pulsa Conectar.

URL del Apps Script
Sin conectar
Código Apps Script a pegar en Google Sheets
const SHEET_NAME = 'Candidatos';

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const data = sheet.getDataRange().getValues();
  const candidates = data.slice(1).map(row => JSON.parse(row[1] || '{}'));
  return ContentService.createTextOutput(JSON.stringify(candidates))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const payload = JSON.parse(e.postData.contents);
  if (payload.action === 'save') {
    const data = sheet.getDataRange().getValues();
    const row = data.findIndex(r => r[0] === payload.candidate.id);
    const values = [payload.candidate.id, JSON.stringify(payload.candidate)];
    if (row > 0) sheet.getRange(row + 1, 1, 1, 2).setValues([values]);
    else sheet.appendRow(values);
  } else if (payload.action === 'delete') {
    const data = sheet.getDataRange().getValues();
    const row = data.findIndex(r => r[0] === payload.id);
    if (row > 0) sheet.deleteRow(row + 1);
  }
  return ContentService.createTextOutput('ok');
}
